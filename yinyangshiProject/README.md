# 阴阳师手游官网 - 静态页面项目

> 唯美奇幻 匠心巨制 — 阴阳师手游官方网站静态复刻版

---

## 📖 项目简介

本项目是**阴阳师手游官方网站**的静态 HTML 页面实现，还原了官网的核心版块布局与视觉风格，包含导航栏、Banner 展示、游戏攻略、同人内容、视听中心、泛娱乐及周边商城等模块。

---

## ✨ 功能模块

- **头部导航**：游戏全目录入口 + Logo + 限时福利领取
- **导航菜单**：官网首页、新闻资讯、式神录、游戏攻略、大社区、泛娱乐、视听中心、主题站、周边商城、藏宝阁等
- **Banner 展示**：官网主视觉大图展示
- **平安世界**：游戏世界观展示板块
- **游戏攻略**：千帝灯斗技攻略、单核SP灯逢魔阵容、入内雀SP花斗技实录等内容卡片
- **同人手账**：同人投稿内容展示
- **同人大触**：式神角色卡片展示（鬼切、大天狗、酒吞童子、晴明、玉藻前等）
- **视听盛宴**：游戏动画视频封面展示
- **泛娱乐**：左右滑动轮播图内容区
- **京都商业街**：可拖动的横向商品展示区

## 🚀 快速开始

### 方式一：直接打开

```bash
# 克隆项目
git clone https://github.com/your-username/yys-website.git

# 进入目录
cd yys-website

# 直接用浏览器打开
open index.html
```

### 方式二：本地服务器（推荐，避免图片跨域问题）

```bash
# 使用 VS Code Live Server 插件，或

# Python 3
python -m http.server 8080

# 然后访问
http://localhost:8080
```

---

## 📌 注意事项

- 本项目为**纯静态页面**，所有链接均为 `#` 占位，不涉及后端接口
- 图片资源需完整保留 `images/` 和 `images1/` 目录，否则图片将无法正常显示
- 建议使用现代浏览器（Chrome、Edge、Firefox）访问以获得最佳效果
- GIF 动图资源较多，首次加载可能稍慢，建议开启本地服务器运行
